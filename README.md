# 🤖 AI Engineer Roadmap

An interactive, visually stunning roadmap to becoming an AI Engineer — built as a single-page web app using **React** (in-browser), **Tailwind CSS**, and vanilla **HTML**.

🌐 **Live Demo:** [vaibhavvvv7.github.io/notai](https://vaibhavvvv7.github.io/notai)

---

## ✨ Features

- **10-Step Structured Roadmap** — curated learning path from Python basics all the way to MLOps and AI deployment
- **Expandable Step Cards** — click any step to reveal its description and hand-picked resources
- **Curated Resources** — each step includes videos 🎥, courses 🎓, blog posts 📝, and tools 🔧 with direct links
- **Portfolio Project Ideas** — 7 real-world AI project suggestions to build and showcase
- **Skills Checklist** — an interactive checklist covering ML, DL, NLP, CV, GenAI, MLOps, and more
- **Dark Mode UI** — sleek dark theme with a polished modern aesthetic

## 🚀 Getting Started

### View Locally

```bash
# Clone the repo
git clone https://github.com/vaibhavvvv7/notai.git
cd notai

# Open in browser — no build step needed!
start index.html   # Windows
open index.html    # macOS
```

### Deploy to GitHub Pages

1. Go to your repo **Settings → Pages**
2. Set the source to the `master` branch, root folder (`/`)
3. Click **Save** — your site will be live at `https://vaibhavvvv7.github.io/notai`
